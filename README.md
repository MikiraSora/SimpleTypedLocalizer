A simple localization library based on SourceGenerator like resX, I designed it and used it for my own AvaloniaBrowser project.

Features:
* support AOT and no reflection need
* reverse out properties from *.resx/*.json though SourceGenerator
* support INotifyPropertyChanged text property for data binding
* append/remove other lang in runtime

Usage:
1. place your lang file like *.resx/*.json into project, and use `<AdditionalFiles>` mark them in your .csproj
<img width="1385" height="499" alt="image" src="https://github.com/user-attachments/assets/90a92a4a-3a39-4475-8c21-120778ac6c81" />

2. define a partial class and add `[ImportLocalizedTextFiles]`:
``` C#
[ImportLocalizedTextFiles("Assets/Languages/lang.json", ImportType.DumpAsStaticDictionaryInCompileTime)]
public partial class MyLang
{
    
}
```

3. just enjoy!
```C#
public static void Main(string[] args)
    {
        var myTextSource = MyLang.B.MyValue;
        myTextSource.PropertyChanged += (s, e) =>
        {
            Console.WriteLine($@"MyValue: {myTextSource.Text}");
        };
        
        LocalizerManager.CurrentDefaultCultureInfo = CultureInfo.InvariantCulture;
        Console.WriteLine($"MyName: {MyLang.MyName}");
        
        LocalizerManager.CurrentDefaultCultureInfo = CultureInfo.GetCultureInfo("zh-CN");
        Console.WriteLine($"MyName: {MyLang.MyName}");
    }
```
